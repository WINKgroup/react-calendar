## Get Started

```
$ npm i wink-calendar
```

## Usage

1. Import the css module
```
import 'wink-calendar/dist/css/style.css';
```

2. Import and use the component
```
import 'wink-calendar/dist/css/style.css';

<Calendar
  selectedDay={date}
  onClick={setDate}
/>
```
