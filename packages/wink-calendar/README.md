```

```

## Usage

1. Import the css module
```
import 'wink-react-calendar/dist/css/style.css';
```

2. Import and use the component
```
import Calendar from 'wink-react-calendar';

<Calendar
  selectedDay={date}
  onClick={setDate}
/>
```
